CSS単位リファレンス

よく使われるCSS単位の一覧です。種類と特徴を確認しましょう。

単位 種類 説明 基準
px 絶対 ピクセル。最も基本的な単位 デバイスピクセル
em 相対 親要素のfont-sizeに対する倍率 親のfont-size
rem 相対 ルート要素(html)のfont-sizeに対する倍率 htmlのfont-size
% 相対 親要素のサイズに対する割合 親要素のサイズ
vw ビューポート ビューポート幅の1% ブラウザの表示幅
vh ビューポート ビューポート高さの1% ブラウザの表示高さ
vmin ビューポート vwとvhの小さい方 幅と高さの小さい方
vmax ビューポート vwとvhの大きい方 幅と高さの大きい方
