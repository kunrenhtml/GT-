CSS Animation 🎨

触って体感する！スライダーを動かしてアニメーションを理解しよう

animation プロパティ
スライダーを動かすとアニメーションがリアルタイムで変化します ✨
💡
animation とは？
CSS だけで要素を動かすための機能です。「どのくらいの時間で」「何回」「どんな速さで」動かすかを細かく設定できます。 下の6つのカードでそれぞれのプロパティを触って確認してみましょう！
animation-duration
⏱ 再生時間を決める。短いと速く、長いとゆっくり動く。単位は秒（s）。
duration 1.0s
.element { animation: spin 1.0s linear infinite; }
animation-delay
⏳ 開始を遅らせる時間。ページ表示後すぐ動かしたくないときに使う。「再生」ボタンで試そう。
delay 0.0s
.element { animation: fadeIn 0.8s ease 0.0s forwards; }
animation-iteration-count
🔁 何回繰り返すかを指定。infinite にすると永遠にループする。ローディングアニメなどに使う。
iteration
.element { animation-iteration-count: 1; }
animation-direction
↔️ 再生の向きを変える。alternate にすると行ったり来たりの自然な動きになる。
direction
.element { animation-direction: normal; }
animation-fill-mode
🎭 終了後の見た目を決める。forwards なら終わった状態のまま止まる（デフォルトは元の位置に戻る）。
fill-mode
.element { animation-fill-mode: forwards; }
animation-play-state
⏸ アニメを一時停止・再開できる。ホバー時に止めたい場合など JavaScript と組み合わせてよく使う。
play-state
.element { animation-play-state: running; }

クリックするとアニメーションが再生されます 👆

@keyframes とは？
アニメーションの「動き方の設計図」です。0%（開始）→ 50%（途中）→ 100%（終了） の各時点でスタイルを指定すると、その間を自動で補間してくれます。 よく使う6パターンをクリックして確認しましょう！

transform とは？
要素の移動・拡大縮小・回転・傾きを指定するプロパティです。アニメーションと組み合わせて使うことが多く、GPU で処理されるためパフォーマンスが良いのが特徴です。 複数の関数を半角スペースで並べて組み合わせることもできます。

animation-timing-function
「一斉スタート」ボタンを押して速度の違いを比べよう 🏁
💡
timing-function とは？
アニメーションの速度の変化パターン（イージング）を指定します。同じ duration でも、このプロパティ次第で動きの印象がまったく変わります。
ease（自然な動き）が最もよく使われ、linear（機械的な動き）、ease-in-out（丁寧な動き）なども頻出です。
cubic-bezier エディター
🎚 自分でイージングを自作できる！4つの数値（0〜1 の制御点）を動かして、スライダーで好みの動きを作ろう。

パフォーマンスのコツ
GPUで処理できるプロパティを使うと 60fps で滑らかになります ⚡
💡
なぜパフォーマンスが大事？
アニメーションは毎秒60回（60fps）画面を更新しています。width や margin をアニメさせると、毎フレームごとにブラウザがレイアウト全体を計算しなおす「Reflow」が発生してカクつきます。 代わりに transform と opacity を使えば GPU が処理してくれるため、常に滑らかに動きます。
⚠️ 重い（避けるべき）
width / height を変化させる

レイアウト再計算（Reflow）が毎フレーム発生。
他の要素も巻き込んで再描画が走る。

/* ❌ 重い */ @keyframes bad { 0% { width: 40px; } 50% { width: 80px; } }
✅ 軽い（推奨）
transform を使用する

GPUの Composite レイヤーで完結。
他の要素に影響せず、60fps を維持できる。

/* ✅ 軽い */ @keyframes good { 0% { transform: scale(1); } 50% { transform: scale(1.3); } }
プロパティ別 GPU 対応
✅ transform（GPU）
✅ opacity（GPU）
✅ filter（GPU）
⚠ width / height（Reflow）
⚠ margin / padding（Reflow）
⚠ top / left（Reflow）
💡 will-change: transform を使おう
/* 事前にブラウザへヒントを渡す */ .animated { will-change: transform, opacity; }
